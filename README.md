# Gestión Biblioteca

Sistema para la gestión de prestamos, miembros y libros en una biblioteca.

## Características principales
- Gestion de libros y miembros
- Registro,devolucion y multa de préstamos
- Administración de multas
- envio de notificacion a miembros registrados
- asignacion de beneficios por mayor cantidad de prestamos a miembros 
- personalizacion de sistema (logo, colores, frase eslogan, multas,beneficios, etc)
